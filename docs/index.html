¿Qué es Heimdall Desktop by Naidel?

Heimdall Desktop by Naidel es una aplicación de escritorio para Windows desarrollada en Python con PySide6.

Permite organizar, lanzar y automatizar aplicaciones, scripts y comandos del sistema desde una interfaz gráfica unificada.

Está pensada para usuarios avanzados, técnicos y administradores que desean control y seguridad sin complejidad innecesaria.

Interfaz principal

La ventana principal muestra un árbol de categorías basado en carpetas reales y un panel de accesos rápidos.

  • Vista por categorías o vista plana
  • Búsqueda instantánea
  • Botones rápidos a carpetas del sistema
  • Integración con menú contextual

Características principales

🔹 Gestión por carpetas: estructura real del sistema de archivos.
🔹 Accesos directos: .lnk, .url, ejecutables, scripts y comandos especiales.
🔹 Botones rápidos: accesos configurables a carpetas frecuentes.
🔹 Scripts: ejecución automática al inicio, salida y mediante tareas.
🔹 Auditoría: registro de acciones y eventos.
🔹 Servidor integrado: HTTP/HTTPS opcional para automatización remota.

Configuración

  • Generales: carpeta base, inicio con Windows, comportamiento al ejecutar.
  • Apariencia: tema, icono, tamaño y posición de ventana.
  • Botones rápidos: configuración de accesos directos.
  • Scripts: scripts de inicio/salida y programador de tareas.
  • Avanzado: importación y exportación de configuración.
  • Log: visor de eventos y gestión de registros.

Servidor HTTP / HTTPS integrado

  • Servidor embebido opcional
  • Soporte TLS con certificados propios
  • Whitelist de rutas y extensiones
  • Autenticación HMAC
  • Backoff por errores
  • Auditoría detallada

Seguridad

  • ✅ Sin ejecución oculta
  • ✅ Control de rutas permitidas
  • ✅ Registro de acciones
  • ✅ TLS y tokens criptográficos


# Release Notes – Heimdall Desktop 0.20.1

---

## Español 🇪🇸

Esta versión introduce una mejora estructural importante en el sistema de configuración,
temas y gestión del servidor.

### Lo más destacado
- Temas totalmente personalizables mediante ficheros QSS.
- Botón Aplicar funcional y coherente en todo el diálogo.
- Importación/exportación de configuración fiable y predecible.
- Refactorización completa de la pestaña Servidor.

Recomendamos esta versión a todos los usuarios.

---

## English 🇬🇧

This release brings major structural improvements to configuration handling,
theme management, and server settings.

### Highlights
- Fully customizable themes using external QSS files.
- Fully functional Apply button across all configuration tabs.
- Reliable and predictable configuration import/export.
- Complete refactor of the Server configuration tab.

This version is strongly recommended for all users.

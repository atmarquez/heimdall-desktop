Configuración · Avanzado

Objetivo: Exportar/Importar configuración y ajustes de registro (log).

Cómo se usa

Exporta el JSON actual o impórtalo desde un archivo.

Elementos y opciones

  • Exportar configuración: Guarda el JSON con todas las preferencias.
  • Importar configuración: Carga un JSON y reemplaza los valores actuales.

Atajos de teclado

  • F1: Ayuda
  • Intro: Aceptar
  • Esc: Cerrar

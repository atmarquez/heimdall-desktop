Configuración · Auditoría

Objetivo: Ver en tiempo real los eventos recientes del servidor embebido.

Cómo se usa

Muestra el contador total y una lista actualizada de los últimos eventos (rolling).

Elementos y opciones

  • Habilitar registro (log): Control duplicado aquí para conveniencia.
  • Fichero de log: Ruta al log; botones para ver/borrar.
  • Lista de eventos: Vista con los eventos más recientes; se refresca periódicamente.

Atajos de teclado

  • F1: Ayuda
  • Intro: Aceptar
  • Esc: Cerrar

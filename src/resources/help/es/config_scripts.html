Configuración · Scripts

Objetivo: Configurar scripts a ejecutar al iniciar/salir de la aplicación.

Cómo se usa

Abre cada diálogo para indicar script, argumentos, si esperar a que termine, timeout y ejecución oculta.

Elementos y opciones

  • Script al iniciar: Acceso al diálogo de configuración de script de arranque.
  • Script al salir: Acceso al diálogo de configuración de script de salida.
  • Resumen: Muestra script+args y opciones (esperar, timeout, oculto).

Atajos de teclado

  • F1: Ayuda
  • Intro: Aceptar
  • Esc: Cerrar

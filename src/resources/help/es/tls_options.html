Diálogo · Opciones TLS

Objetivo: Seleccionar certificado, clave privada y versión mínima de TLS para HTTPS.

Cómo se usa

Explora los archivos PEM/CRT/KEY y elige la versión mínima (TLS 1.2 o 1.3). Puedes generar un certificado local autofirmado.

Elementos y opciones

  • Certificado (PEM): Archivo de certificado en formato PEM/CRT.
  • Clave privada: Archivo de clave privada (KEY/PEM).
  • Versión mínima TLS: TLS 1.2 o 1.3.
  • Generar certificado local: Crea cert.pem y key.pem en resources/ssl/ para uso local.

Atajos de teclado

  • F1: Ayuda
  • Intro: Aceptar
  • Esc: Cancelar

Diálogo · Importar elementos (arrastrar/soltar)

Objetivo: Elegir si crear accesos (.lnk) o copiar físicamente los elementos arrastrados.

Cómo se usa

Cuando sueltas archivos/carpetas en el árbol, este diálogo permite elegir entre “Crear acceso directo” o “Copiar”.

Elementos y opciones

  • Crear acceso directo: Genera un .lnk apuntando al original (solo Windows).
  • Copiar el elemento: Copia el archivo o la carpeta dentro del destino.

Atajos de teclado

  • F1: Ayuda
  • Intro: Aceptar
  • Esc: Cancelar

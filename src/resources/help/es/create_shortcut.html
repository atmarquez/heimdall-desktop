Diálogo · Nuevo acceso directo (.lnk)

Objetivo: Crear un acceso directo de Windows a un ejecutable con argumentos e icono.

Cómo se usa

Selecciona el programa de destino (.exe), argumentos opcionales, carpeta de trabajo e icono.

Elementos y opciones

  • Nombre del acceso: Texto con el que se listará el .lnk.
  • Programa (destino): Ruta al ejecutable a lanzar.
  • Argumentos (opcional): Parámetros para el ejecutable.
  • Iniciar en: Carpeta de trabajo al ejecutar.
  • Icono: Icono a mostrar; puede ser .ico o extraerse de .exe/.dll.

Atajos de teclado

  • F1: Ayuda
  • Intro: Crear
  • Esc: Cancelar

Diálogo · Opciones del servidor

Objetivo: Configurar puerto, alcance (local/red), lista blanca de extensiones y notificaciones.

Cómo se usa

Introduce el puerto, si solo debe aceptar peticiones locales, activa “Whitelist carpeta base”, define extensiones permitidas y el modo de notificaciones.

Elementos y opciones

  • Puerto de escucha: Número de puerto TCP (1–65535).
  • Atender solo peticiones locales: Restringe a 127.0.0.1 / ::1.
  • Whitelist carpeta base: Obliga a rutas relativas respecto a la carpeta base.
  • Extensiones permitidas: Lista separada por comas, con punto (por ejemplo .exe,.cmd,.ps1).
  • Notificaciones del servidor: Todas, Ninguna, Solo correctas u Solo errores.

Atajos de teclado

  • F1: Ayuda
  • Intro: Aceptar
  • Esc: Cancelar

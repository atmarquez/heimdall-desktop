Diálogo · Auditoría del servidor

Objetivo: Visualizar los últimos N eventos del servidor con actualización periódica.

Cómo se usa

Elige cuántos eventos ver y revisa el contador total. Sirve para diagnosticar llamadas y errores.

Elementos y opciones

  • Mostrar últimos N: Cantidad de eventos recientes a listar.
  • Lista de eventos: Vista tipo log con hora, acción y parámetros relevantes.

Atajos de teclado

  • F1: Ayuda
  • Esc: Cerrar

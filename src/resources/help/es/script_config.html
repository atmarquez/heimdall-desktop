Diálogo · Script de inicio/salida

Objetivo: Definir un script a ejecutar antes de abrir la app o al salir, con argumentos y control de espera.

Cómo se usa

Selecciona el script, opcionalmente parámetros, define si esperar a su finalización y el timeout. En Windows puedes ejecutarlo oculto.

Elementos y opciones

  • Ruta del script: Archivo .bat/.cmd/.ps1/.py/.vbs/.exe, etc.
  • Argumentos: Línea de argumentos tal cual se pasarán.
  • Esperar a que termine: Si está activo, la app espera a que el script termine.
  • Timeout (s): 0 para sin límite.
  • Ejecutar oculto (Windows): Oculta la ventana de consola/PowerShell.

Atajos de teclado

  • F1: Ayuda
  • Intro: Aceptar
  • Esc: Cancelar

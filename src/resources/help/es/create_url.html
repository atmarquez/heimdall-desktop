Diálogo · Nuevo acceso web (.url)

Objetivo: Crear un acceso directo a una página web.

Cómo se usa

Introduce un “Nombre” y la “URL” con esquema (https://...). Puedes pegar desde el portapapeles y el sistema sugerirá un nombre basado en el dominio.

Elementos y opciones

  • Nombre: Título del acceso que se verá en el árbol.
  • URL: Dirección completa con esquema: p. ej., https://sitio.com
  • Pegar desde portapapeles: Rellena la URL con el texto del portapapeles si parece una dirección.

Atajos de teclado

  • F1: Ayuda
  • Intro: Crear
  • Esc: Cancelar

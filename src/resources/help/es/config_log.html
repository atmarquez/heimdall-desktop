Configuración · Log

Objetivo: Exportar/Importar configuración y ajustes de registro (log).

Cómo se usa

Configura la ruta del log, ábrelo o bórralo.

Elementos y opciones

  • Habilitar registro (log): Activa/desactiva escritura al fichero de log.
  • Fichero de log: Ruta del archivo donde se escriben los eventos.
  • Ver/Borrar log: Abre el archivo o limpia su contenido.

Atajos de teclado

  • F1: Ayuda
  • Intro: Aceptar
  • Esc: Cerrar

Configuración · Generales

Objetivo: Definir carpeta base, comportamiento de inicio, título y preferencias generales.

Cómo se usa

Ajusta la carpeta base donde se guardan categorías/accesos, define cómo se muestra la app al iniciar y si debe minimizarse a la bandeja. Configura el título y el motor de búsqueda web.

Elementos y opciones

  • Carpeta base de categorías: Ruta raíz donde se organizan las carpetas (categorías) y accesos.
  • Nombre para accesos en raíz: Etiqueta para elementos ubicados directamente en la carpeta base.
  • Título de la aplicación: Texto mostrado en la ventana y en la bandeja del sistema.
  • Iniciar minimizado a la bandeja: Oculta la ventana al arrancar y deja el icono en la bandeja.
  • Minimizar a la bandeja al cerrar: Evita salir por error; al cerrar la ventana, se minimiza a la bandeja.
  • Ocultar al lanzar una aplicación: Tras ejecutar un acceso, la ventana se oculta automáticamente.
  • Al iniciar: Modo de vista inicial: agrupada, plana, expandir todo o contraer todo.
  • Mostrar al pulsar el icono de bandeja: Dónde debe aparecer la ventana (esquina inferior derecha o junto al cursor).
  • URL para "Buscar en la web": Define el motor de búsqueda que se abrirá desde el panel derecho.

Atajos de teclado

  • F1: Ayuda
  • Intro: Aceptar
  • Esc: Cerrar

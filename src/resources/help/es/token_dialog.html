Diálogo · Calcular Token

Objetivo: Generar un token HMAC válido para el servidor HTTP/HTTPS integrado, basado en la clave secreta configurada.

¿Qué es un token?

El servidor puede requerir un token HMAC-SHA256 para autorizar la ejecución remota de aplicaciones mediante URL.
El token garantiza que solo usuarios que conocen la clave secreta puedan lanzar accesos.

Cómo se usa

Introduce o confirma los valores necesarios y pulsa Calcular. El diálogo mostrará tanto:

  • El token generado
  • Una URL completa de ejemplo con el token aplicado
  • Opciones para copiar el token o la URL al portapapeles

Elementos y opciones

  • Habilitar protección por token: Activa o desactiva la validación de token en el servidor.
  • Clave secreta protegida (DPAPI): La clave se guarda cifrada en Windows mediante DPAPI. El diálogo permite:
    • Ver si existe clave cargada
    • Reemplazarla por una nueva
    • Importar una clave existente
  • Ruta de la aplicación / Parámetros: El diálogo puede mostrar (o permitir editar) la ruta respecto a la cual se calculará el token de ejemplo.
  • Cálculo del token: Genera el HMAC usando: HMAC_SHA256(secret, path).
  • Token resultante: Campo de solo lectura con el token generado.
  • URL generada: Muestra una URL del tipo:
    http(s)://127.0.0.1:8080/?path=RutaApp&token=XXXX
  • Botones “Copiar”: Copia el token o la URL al portapapeles.

Atajos de teclado

  • F1: Ayuda
  • Intro: Aceptar / Calcular
  • Esc: Cancelar

Notas adicionales

  • El token cambia si cambia la clave secreta o la ruta usada en la firma.
  • Si la opción “Whitelist carpeta base” está activa en el servidor, la ruta debe ser relativa a la carpeta base configurada.
  • DPAPI solo está disponible en Windows.

Configuración · Servidor

Objetivo: Habilitar el servidor HTTP/HTTPS embebido y configurar opciones resumidas.

Cómo se usa

Desde esta pestaña puedes abrir los diálogos de “Opciones del servidor”, “Opciones TLS”, “Backoff (errores)” y el visor de “Auditoría…”.

Elementos y opciones

  • Habilitar servidor: Activa/desactiva el servidor embebido.
  • Configuración de servidor…: Abre el diálogo con puerto, alcance, lista blanca y notificaciones.
  • TLS…: Abre el diálogo para cert/clave y versión mínima.
  • Backoff…: Configura ventana/umbral y retrasos ante errores repetidos.
  • Auditoría…: Abre el visor con los últimos eventos registrados.

Atajos de teclado

  • F1: Ayuda
  • Intro: Aceptar
  • Esc: Cerrar

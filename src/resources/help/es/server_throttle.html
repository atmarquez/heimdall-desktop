Diálogo · Backoff del servidor

Objetivo: Configurar el retardo progresivo ante múltiples errores desde el mismo origen (IP/Agente).

Cómo se usa

Define ventana de conteo en segundos, retardo base/máximo en milisegundos y a partir de cuántos errores aplicar penalización.

Elementos y opciones

  • Ventana de conteo de errores (s): Intervalo temporal donde se acumulan los errores.
  • Retardo base (ms): Retraso inicial cuando se supera el umbral.
  • Retardo máximo (ms): Límite superior del retraso.
  • Umbral de errores: Errores en ventana a partir de los cuales se penaliza.

Atajos de teclado

  • F1: Ayuda
  • Intro: Aceptar
  • Esc: Cancelar
